# Smart Home :house:
This project involves a smart home system used to control multiple utilities using a mobile application. :iphone:
> **Hardware Used**: GSM SIM808 Module, Heater, Fan, Led.

> **Software Ussed**: Flutter (Dart), Arduino C/C++.
